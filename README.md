# MDA Microplan Tool

A browser-based tool for Mass Drug Administration (NTD) microplanning: projected
population, MLoS-based microplan, NPC rationalization with surplus/deficit
redistribution, summary exports, and daily implementation planning.

Open `index.html` in a browser, or use the hosted GitHub Pages link.

The tool runs entirely in the browser — no server or database. The XLSX, JSZip and
html2canvas libraries are bundled in `vendor/` so it works offline.
